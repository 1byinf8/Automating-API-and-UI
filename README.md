# Automation Assignment — Playwright + POM

## Framework Used
- Playwright (TypeScript)
- Page Object Model (POM)
- API Testing with Playwright

## How to Install
```bash
npm install
npx playwright install
````

## How to Run All Tests

```bash
npx playwright test
```

## Project Structure

* `/pages` → Page Object Models
* `/tests` → UI + API Tests
* `/utils` → API helper functions

## Use Cases Covered

### 1. Message Box Task (UI Automation)

* Login → Create Task Bot → Add Message Box → Save

### 2. Form With Upload (UI Automation)

* Login → Create Form → Drag textbox + upload → Fill → Upload → Save

### 3. Learning Instance API Automation

* Login API
* Create Learning Instance API
* Validate response fields
